# Habit Tracker

A web app that helps you track good habits

- [Live preview](https://zesty-kheer-900272.netlify.app/)

## Features

- Add habits you want to cultivate
- Track you habits over time with week visualization
- Habit streaks

## Technologies used

- [React](https://react.dev)
- [tailwindcss](https://tailwindcss.com/)
- [date-fns](https://date-fns.org/)
